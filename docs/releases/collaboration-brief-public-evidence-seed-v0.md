# Collaboration Brief: Public Evidence Seed v0

## What This Release Is

`Public Evidence Seed v0` is the first bounded public release from Alzheimer's
Open Intelligence, a living release series focused on building a trustworthy
Alzheimer's evidence substrate from official public sources.

## What Is In The Release

- normalized seed dataset artifacts
- cross-source linkage artifacts
- exact-anchor intervention review records
- watchlist review records for unresolved candidates
- selected scripts and schemas for reproducibility

## Why This Is Useful

The release gives researchers a tractable public layer for:

- intervention discovery
- grant-publication-trial linkage
- public evidence review
- machine-readable reuse

## What Collaborators Can Do With It

- inspect and challenge review decisions
- extend the extraction and linkage logic
- add biomarker-family evaluation surfaces
- propose additional public-source integrations
- use the release as a starting point for downstream methods work

## What This Is Not

- not a patient-level cohort dataset
- not a treatment recommendation engine
- not a claim to have mapped all Alzheimer's evidence
