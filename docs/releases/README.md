# Releases

This directory holds versioned public scientific releases for Alzheimer's Open
Intelligence.

The release model for this repository is:

- bounded
- citeable
- reproducible
- cumulative

Each release should make one strong exact claim, link the relevant artifacts
directly, and preserve a stable record of what the repository contained at that
moment.
