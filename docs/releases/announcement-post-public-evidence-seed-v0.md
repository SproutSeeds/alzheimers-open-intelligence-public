# Announcement Post: Public Evidence Seed v0

Alzheimer's Open Intelligence is releasing `Public Evidence Seed v0`, a
public-record-derived Alzheimer's evidence substrate built from official public
sources including ClinicalTrials.gov, PubMed, PMC Open Access, NIH RePORTER,
and openFDA.

This release is meant to make basic Alzheimer's evidence work easier to
inspect, compare, and reuse.

What is included:

- a linked seed dataset
- explicit linkage artifacts
- exact-anchor intervention review records
- watchlist review records for unresolved candidates
- selected scripts and schemas for reproducibility

Current validated snapshot:

- `105` public-record seed rows
- `5` official public source systems
- `4` exact-anchor review records
- `2` watchlist review records

This is not a patient-level cohort release and not a clinical decision tool.
It is a bounded public evidence release designed for research use, inspection,
and extension.
