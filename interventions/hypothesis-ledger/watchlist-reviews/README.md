# Watchlist Reviews

This directory holds completed review packets for candidate interventions that
remain unresolved after review.

Use this layer to:

- preserve the strongest current public support without overstating it
- make blockers explicit for unresolved candidates
- keep next actions attached to the object rather than buried in a rolling report

This directory should not be used to quietly broaden an intervention claim.
Watchlist review packets exist so unresolved candidates stay legible and honest
until they either earn stronger support or are retired.
