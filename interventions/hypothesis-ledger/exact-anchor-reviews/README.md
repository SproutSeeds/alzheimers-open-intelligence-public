# Exact Anchor Reviews

This directory holds completed review packets for interventions that reached the
exact-anchor promotion queue from the public-record linkage lane.

Use this layer to:

- decide whether a promotion candidate should enter the exact-anchor surface
- separate qualifying exact-scope support from related but non-qualifying records
- preserve caution notes and next actions alongside the decision

This directory should not silently promote an intervention just because it has a
high candidate score. The exact-anchor review packet is where that narrower
judgment is made explicitly.
