# Public Record Derived

This directory is for Alzheimer's-specific datasets that we derive from public
records rather than controlled participant-level cohorts.

The first active object is a structured intervention and biomarker evidence
dataset built from official public sources such as:

- ClinicalTrials.gov
- PubMed
- PMC Open Access Subset
- NIH RePORTER
- openFDA

This directory is for:

- dataset briefs
- source registries
- query manifests
- templates
- later, derived non-secret dataset snapshots

This directory is not for:

- raw controlled cohort dumps
- scraped private content
- opaque one-off notebooks without contracts
